TWO LAYER SECURITY SYSTEM

A Double Layer Security System provides extra protection using two authentication methods, like PIN with RFID, fingerprint, or OTP. Only when both verifications match, access is granted; otherwise, an alarm is triggered. It is widely used in doors, lockers, ATMs, and smart devices for higher safety.

BATTERY LEVEL INDICATOR

A Battery Level Indicator monitors battery charge by measuring voltage and displaying it through LEDs or an OLED/LCD. Using a voltage divider and microcontroller/IC, the voltage is mapped to levels (25%, 50%, 75%, 100%), with alerts for low or full charge. It is useful in devices like power banks, emergency lights, and solar systems, with scope for IoT-based mobile notifications.
